# Kiwi → Qase Migration Tool

Tool for migrating test cases from Kiwi to Qase with validation, diff preview and audit logging.

---

## 🚀 Features

- Kiwi test case parser
- Step + precondition extraction
- Qase API integration
- Payload validation layer
- Diff preview before update
- Audit logging (JSONL)
- 3 migration modes (1/2/3)
